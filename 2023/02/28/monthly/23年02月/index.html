23 年 02 月

从各个地方看到并记录下来的一些内容。

资源整合

学习

  1. 关于自定义 Hooks，是什么、可复用的提取、不可复用的提取、提取数量等，最后还是要注意抽象的代价，抽象不是免费的，有时适当重复反而更好
  2. 如何处理 React 中的错误
  3. 《Modularizing React Applications with Established UI Patterns》。
    1. 作者通过案例教你如何用成熟的 UI 模式把 React 应用模块化。说人话就是，把大组件拆小。几个思路: 1）提取子组件以拆分视图，2）提取非视图逻辑到 hooks，3）提取领域模型以封装逻辑。但是也要注意别过度优化。

工具

others

  1. 2022 大前端总结&2023 就业分析
    1. “少抱怨，多思考，未来更美好”，社会需要专业人才，喜欢就坚定的做
    2. 前端各种框架百花齐放，2020 年之后逐渐趋于成熟，开始卷深度、卷体验，人要随着行业发展的洪流走

Some statements

  1. 管理的精髓就是洞察人性和接受不完美
  2. 一只站在树上的鸟从来不会担心树枝断裂，因为它相信的不是树枝，而是自己的翅膀。
  3. 要是有人问你，你愿意做快乐的猪呢，还是愿意做苦恼的哲学家？你就回答：我愿意做快乐的哲学家，这样可以显出你的聪明。
  4. 当你手上拿着一杯水，接下来你要做什么？做你自己该做的事情，和水没关。
  5. 受到的教育一直是不管做什么事情都要从中获得什么，哪怕是看个电视
  6. 我来这个世界不是创造完美的，是来经历的
  7. 当你全身心做一件事情的时候，是感觉不到时间流逝的，你只是做，像个机器人，想不起吃饭、睡觉、排泄、娱乐，这种经历一辈子有几次呢？有些人，可能一辈子都不会有一次
  8. 在对下的沟通交流上，不要指望着下级主动；在对上的沟通交流中，作为下级要找时机主动 –当你上下级都体验过之后
